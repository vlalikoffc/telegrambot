# FAQ

### Можно ли использовать Telegram API напрямую?
Нет. Плагины не должны обращаться к Telegram API.

### Можно ли изменять статус?
Да, только через `ctx.status` в `on_render()`.

### Можно ли читать другие плагины?
Можно читать файлы, лежащие прямо в `plugins:/` (только чтение).  
Чужие папки плагинов (`plugins/<other_plugin>/`) недоступны.

### Можно ли хранить данные?
Да. Используйте `ctx.storage` или файлы в `self:/`.

### Можно ли делать библиотеки для других плагинов?
Да, но только если они расположены в `plugins:/` и используются как read‑only.
